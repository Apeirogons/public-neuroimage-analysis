Google Colab code for running a classification algorithm using 3D CNNs on magnetic resonance imaging data to predict gender.
heatmap_maker.ipynb generates feature importance heatmaps based on an implementation of Integrated Gradients found elsewhere on GitHub.

Data taken from the AOMIC neuroimage dataset, which can be downloaded using download_aomic.sh.

Note that due to GPU VRAM limitations, the batch size is set to 1.
